# Digital Workshop

Готовый одностраничный сайт-портфолио freelance-разработчика. Работает без сборщика и внешних зависимостей.

## Запуск

Откройте `index.html` двойным кликом или запустите локальный сервер:

```powershell
python -m http.server 4173 --directory .
```

Затем откройте `http://localhost:4173`.

## Что заменить перед публикацией

1. При необходимости обновите имя, email и ссылки на соцсети в `index.html`.
2. Telegram username для формы задаётся в `script.js` и в `action` формы в `index.html`.
3. Три демонстрационных кейса — на реальные проекты. Тексты модальных окон находятся в объекте `caseData` в `script.js`.
4. После появления реальных кейсов добавьте для них отдельные индексируемые страницы.

Форма проверяет обязательные поля, собирает текст заявки и открывает личный чат Telegram с готовым черновиком. Сайт не хранит данные и не зависит от почтового посредника. Для полноценной серверной формы потребуется подтверждённый провайдер или serverless-функция с серверной антиспам-защитой.

В production подключены `styles.min.css` и `script.min.js`; читаемые исходники остаются в `styles.css` и `script.js`. После изменения исходников пересоберите минифицированные файлы.

## SEO и безопасность

- `robots.txt`, `sitemap.xml`, canonical, Open Graph, Twitter Cards и Schema.org добавлены.
- CSP ограничивает скрипты, стили, соединения и отправку форм.
- Политика конфиденциальности находится в `privacy.html`, контакт для security-отчётов — в `.well-known/security.txt`.
