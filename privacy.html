Политика конфиденциальности

Здесь простым языком описано, что происходит с данными, когда вы связываетесь со мной через этот сайт.

Какие данные используются

Форма запрашивает имя, тип проекта, описание задачи и, по желанию, дополнительный контакт. Не указывайте в заявке пароли, реквизиты карт, документы, медицинские или другие чувствительные сведения.

Как работает форма

Сайт не отправляет заявку на собственный сервер, не записывает её в базу и не хранит в браузере. После нажатия «Продолжить в Telegram» браузер передаёт введённый текст сервису Telegram как черновик сообщения. Сообщение будет отправлено мне только после того, как вы нажмёте «Отправить» в Telegram.

Цель и срок хранения

Полученные сообщения используются только для ответа на обращение, оценки задачи и дальнейшей работы по вашему запросу. Переписка может храниться в Telegram или почте, пока она нужна для общения и выполнения договорённостей. Вы можете попросить удалить её, написав мне.

Сторонние сервисы

Сайт размещён на GitHub Pages, поэтому GitHub может обрабатывать стандартные технические данные запросов, например IP-адрес и сведения браузера. При переходе в Telegram, MAX, GitHub или почтовое приложение действуют правила соответствующего сервиса.

  • Политика конфиденциальности Telegram

Cookies и аналитика
